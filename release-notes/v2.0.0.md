# Release Notes - v2.0.0

**Release Date**: 2026-04-06

## 🎯 Summary

First production release of AI JIRA Automation - a professional system for creating JIRA Epics, Stories, and Subtasks from implementation plans.

**Impact**: New system  
**Breaking Changes**: N/A (Initial release)  
**Migration Required**: No

---

## ✨ Features

### Core Automation
- **Epic Creation**: Interactive creation of JIRA Epics with comprehensive templates
- **Story Creation**: Create Stories with user story format and acceptance criteria
- **Subtask Creation**: Break stories into implementation tasks
- **Automatic Linking**: Epic → Story → Subtask hierarchy maintained automatically

### Intelligent Features
- **Smart Epic Linking**: Automatically suggests relevant epics based on keywords
- **Epic Search**: Find and link to existing epics in your component
- **Field Validation**: Built-in validation for required fields
- **Preview Mode**: Review issues before creation

### Templates
- **Professional Templates**: JIRA wiki markup with comprehensive sections
  - Epics: 11 sections (Overview, Business Value, Goals, etc.)
  - Stories: 8 sections (User Story, Acceptance Criteria, etc.)
  - Subtasks: 7 sections (Implementation Steps, Definition of Done, etc.)
- **Customizable**: Edit YAML templates to match your workflow

### Configuration
- **Flexible Setup**: YAML-based configuration
- **Custom Fields**: Configurable custom field mappings
- **Default Values**: Sensible defaults for priority, component, labels

### Interfaces
- **CLI Commands**: Direct command-line interface
- **Interactive Mode**: Prompted input with validation
- **Natural Language**: Integration with Claude Code for natural language commands

### Developer Experience
- **Error Handling**: Clear error messages
- **Audit Trail**: JSON output for all created issues
- **Help System**: Built-in help and usage documentation
